# MyCine

Sistema inteligente de recomendação de filmes desenvolvido em Java com foco em testes unitários utilizando JUnit 5 e Mockito.

1. Integrantes

- Igor Gabriel
- Karen Ferreira

2. Como rodar o projeto

1. Abrir o projeto no Eclipse
2. Executar a classe:

- service.Main

3. O sistema abrirá no console permitindo informar:
- Nome
- Idade
- Idioma
- Duração
- Gênero favorito
- Notificações

Após isso, o sistema gera recomendações personalizadas.

4. Como rodar os testes

- Clique direito no projeto
- Selecione:

Run As → JUnit Test

Atualmente o projeto possui:

- 43 testes automatizados
- 0 falhas
- 0 erros
- Cobertura JaCoCo de 85%

5. Tecnologias utilizadas

- Java
- JUnit 5
- Mockito
- Eclipse IDE

6. Estrutura do projeto

- model = entidades do sistema
- service = regras de negócio
- exception = exceções específicas
- util = utilitários auxiliares
- testes = testes unitários
